## 一、章节定位与魂

**魂（纲领裁定，不可更改）**：礼＝陌生人之间可持续协作的「接口协议」——让互不相识、彼此低信任的人可预测地互动，把摩擦降到不需要靠暴力或权威压制的程度。

- 内核取「克己、敬、分寸」一路，不取「别序 / 等级秩序」一路。
- 全书弧线中的位置：礼是**射的日常反面**。射证明「从个体胜任到陌生人组织」这种能力真实存在（危机把利害强行对齐）；礼回答「危机临时供给的东西，日常到底缺了什么」（日常里利害各自散着，没有外力替你对齐）。礼是**全书的日常智识核心**，可以不是最长，但不能被边缘化。
- 声部：社会 / 日常声部（非身体 / 危机）。同时是纲领「免疫渐强线」的**起点**——礼把「维系合作」的机制从「靠地位」换成「靠平等」，读者的抗裹挟免疫在这一章就已点亮（详见纲领第八节第 7 点）。

**一句话说清「这一章给读者什么本事」**：

> 让你在没有熟人、没有规则强制、没有人替你撑腰的场合，也能和陌生人低摩擦地对上话、搭起够用的信任、把冲突压下去、把一件共用的事维持住。

## 二、目标读者与深度

### 2.1 受众基线（按最新 AGENTS.md §1 重设，与旧版不同）

- **主要读者：16–24 岁**；其次 10–18 岁。
- **深度天花板按 16–24 岁设定，不为迁就低龄读者而降低深度。** 旧大纲把礼定位为「面向 10–18 岁全年龄段」，各节例子与练习默认收缩到中小学尺度——本版取消这一默认。例子取材扩展到兼职、合租、志愿服务、线上社群、社团、初入职场这类 16–24 岁的真实场景；10–18 岁的调整以「补充说明」方式处理，不反向拉低整章深度。
- 面向低龄读者的适配，落在「同一机制换一个更小尺度的例子」，不落在「把机制讲浅、讲粗」。

### 2.2 深度靠什么承载

礼作为全书日常智识核心，深度要顶到 16–24 岁水平，但深度由三样东西承载，**专业公式与学术 notation 一律留在 `knowledge/`，正文不出现**：

1. **白话机制**：把「为什么会这样」讲成一条能复述的因果链，不用术语堆砌。
2. **具体数字**：LinkedIn 两千万用户随机实验的倒 U 型效应、最后通牒博弈里低于两成的报价约一半被拒、Ostrom 八条特征里最强的一条与最弱的一条差了近六倍、聚光灯效应中「预测约 46% 会被注意、实际约 23%」、一周和陌生人说话练习里「87% 是跟接近的第一个人就谈成的」——数字进正文，是「白话机制 + 具体数字」的具体数字。
3. **真实案例**：都是普通人做日常小事的场景，不制造英雄。

**移入 `knowledge/` 不进正文的东西**：条件概率记号、p 值、置信区间、统计模型名称（贝叶斯多层模型 / PGLS 等）、ratio 数值、学术框架的原始名称（「组织信任模型」「四成分模型」「设计原则」这类清单式标签）。正文里，「支持力度只勉强到统计上『有点意思』的水平」替代「p=0.065」；「维系稳定关系的数量有上限，但具体是十几人还是几百人，学界分歧很大」替代「邓巴数 150 / 置信区间 2–520」。

### 2.3 一个深度上的主动选择：证据诚实本身就是深度

礼要顶到的深度里，有一层是「哪些说法是硬的、哪些是软的、哪些流行说法其实站不住」。戈特曼「90% 预测离婚」被剔除、罗伯斯山洞实验被曝出人为操纵、邓巴数的巨大统计争议、Axelrod「一报还一报」高度依赖比赛格式——这些「拆穿流行说法」的内容不藏起来，正文里如实讲。它既是深度，也顺带兑现纲领的「免疫渐强」：一个会追问「这个数字是怎么来的」的读者，正是礼想培养的读者。

## 三、叙述顺序与各节要点

### 3.0 全章暗线（肯定式表述，替代旧版的对比否定表述）

危机能把陌生人临时拧成一股绳，靠的是生死把所有人的利害对齐了。日常没有这种外力，利害各自散着。礼这一整套接口，就是在「利害不会自动对齐」的常态里，一点一点把摩擦降到不需要暴力和权威压制的水平。五个子章节是这件事的五种具体机制。

（写作纪律：这条暗线在正文中用肯定句陈述。禁止写成「礼要治的不是危机而是日常」「靠的不是外力而是……」这类对比否定。）

### 3.1 每节统一的叙述流（按最新 AGENTS.md §2，取代旧版「四条逻辑功能规范」的抽象表格）

每一节按同一条五步流推进，**收尾一律落在读者自身处境与本周可做的一步，不升华、不布道**：

1. **真实案例**：从读者上个月可能真遇到过的平淡社交场景切入（不是虚构危机）。
2. **打破直觉——让数据自己说**：先摆案例与一个具体的、反直觉的数字，让「原来不是我以为的那样」从事实里自己浮现。**不写「你以为 A，其实是 B」这种先否定读者再纠正的句式**，只摆事实和数字，结论读者自得。
3. **权威框架**：给出机制的白话解释与其来源、以及它的边界（哪里成立、哪里不成立）。
4. **细颗粒操作**：一个本周就能做、低风险、可观察、不要求读者组织任何人的具体动作。
5. **回到读者自身处境**：说明这一样你其实已经在做一部分，指出一个下一步。语气是「你可以」，不是「你应该」；给选项，不下命令。

### 3.2 五个子章节

> 五个主题与其顺序由纲领第四节锁定。递进：**社会的原料 →（一条链怎么搭起来）→（一条链受压怎么不断）→（维持一条链最省力的杠杆）→（这一切在群体尺度怎么涌现）**。所有子章节标题与开篇一律用肯定式。

#### 子章节一：弱联系与互惠——社会的承重结构

- **① 真实案例**：上一次找到实习、兼职、租房线索或一条要紧信息，消息其实来自谁——多半是一个不常联系的熟人、一个点头之交，而不是天天在一起的死党。
- **② 打破直觉（数据自现）**：LinkedIn 对超两千万用户做了五年随机对照实验，随机改变人们网络里弱联系的多少，结果弱联系确实带来更多新工作；但效应是**倒 U 型**——中间强度的联系最有用，太强太弱都差一些，而且这效应随行业数字化程度变化。格兰诺维特 1970 年代对 282 名求职者的调查，早就发现更多人靠弱联系而非强联系找到工作。
- **③ 权威框架**：弱联系连接了不同的社交圈，能带来你自己圈子里没有的「非冗余信息」，它桥接的是「结构洞」。互惠是几乎所有社会都共有的一条规范（先收了别人的好意，就欠着一份还的义务）。不同关系距离，对应不同的互惠期待：亲近的人之间是「广义互惠」（不必即时对等），陌生人之间是「平衡互惠」（期待一段时间内大致扯平）——这解释了为什么把亲近关系每一笔都算清反而伤感情。
- **④ 细颗粒操作**：维护一个「熟悉的陌生人」——记住一个名字、多问一句、还一次小人情。先给出去一点点（对陌生人用平衡互惠：一个举手之劳，不指望立刻回报）。**对无法 / 不愿开口的读者的最低档**：成为一个可靠的常在身影——单是常出现在同一个场合、被对方认出来，就已经在积累好感（重复曝光效应），不需要说一句话；或者维护线上一个熟人账号的往来。
- **⑤ 回到读者**：你身边其实已经有若干条这样的弱连接。本周挑一条，做一个具体动作（记住一个名字、还一个小人情）。城市里大量弱连接织成的网，是信息和日常安全感的来源；你多稳住一条，网就多一个承重点。
- **本节红线提示**：邓巴数只讲「维系稳定关系的数量有上限，但具体数值争议很大」，不给「150」这个精确数字；Regan「可乐实验」只作历史图示，标注样本量待证实、无独立复现；倒 U 型必须写出，不能写成「越弱越好」；弱联系按定义就是低频低强度，不预设读者外向，内向者在此有天然优势。

#### 子章节二：信任的分层搭建——一格一格加载

- **① 真实案例**：要不要相信一个刚认识的人——借他一样东西、跟他合作一件事、把一件小事托付给他——那种「信又不太敢信」的纠结。
- **② 打破直觉（数据自现）**：信任可以是一条进度条，一格一格加载，而不是一个「信 / 不信」的开关。投资博弈实验里，人们愿意先拿出一部分钱交给素不相识的对方（对方拿到时钱会被放大），对方也确实会返还一部分——「信任」和「回报」是人类行为里稳定存在的两样东西；一项汇总了 162 次复现、逾两万三千名参与者的分析显示，信任水平在不同文化 / 地区之间存在系统性差异。
- **③ 权威框架**：把信任拆成两种——「能力信任」（他能不能把事做成）和「品格信任」（他会不会坑我）。在纯人际关系里，品格（善意 + 正直）比能力更主导（一项对 220 名大学生友谊的研究里，能力对朋友间的信任几乎不起作用）。还有一种「临时信任」：几分钟内靠角色、规则和共同任务就搭起够用的信任（一起做小组作业、一起值一班志愿岗），它脆弱、依赖外部结构，出现负面信号容易崩，不是长期信任的终点。迭代相处里，「先伸手示好、对方坑你就回应、对方回头就原谅、态度清晰」是个稳的直觉——但要如实说明：这来自计算机策略比赛，真人里约七成实际用的是「赢了就接着来、输了就改」，不能当人际处方硬套。
- **④ 细颗粒操作**：做一次「小额信任试探」——把一件低风险的小事托付给还不熟的人，观察结果。把信任当进度条、不一次给满。**对曾被辜负 / 被霸凌的读者**：先小步、可逆、可观察地试探，正是保护你自己的方式，不是心眼小、不是软弱；信任确实更容易被打破、更难被建立，但这不是铁律，很大程度上取决于具体情境。
- **⑤ 回到读者**：你其实一直在用小步试探这套办法。本周挑一件低风险的事，试一次。
- **本节红线提示**：Axelrod「一报还一报」必须两面呈现（比赛理论结果 + 真人系统性偏离），不单向包装成人际处方；迈耶模型完全不提「组织信任」原名，只讲「能力信任 vs 品格信任」；快速信任必须标注脆弱性；信任博弈标注为「实验室投资博弈」这一具体范式；给被辜负读者的重构不能写成「信任一旦破了就再也建不回来」。

#### 子章节三：冲突降级与分寸——克己作为一种技艺

- **① 真实案例**：那次吵架，是从哪一句话开始收不住的。
- **② 打破直觉（数据自现）**：大多数冲突升级，是节奏失控，跟谁对谁错关系不大。当生理唤醒冲得太高，人处理信息的能力会明显下降——这有独立的神经科学证据（急性、不可控的应激会迅速削弱前额叶皮层的功能，人转向更原始的反应），也有唤醒—表现关系的经典曲线支持，不依赖任何有争议的婚姻预测数据。
- **③ 权威框架**：「面子」是互动中被动态维持的正向自我形象，日常交往里人们靠一整套礼节性动作互相保面子、避免让彼此难堪（面子工作，本节最稳的地基）。礼貌行为很多是在补救「威胁到对方面子」的举动——不过要加注：这套「正 / 负面子」框架源自英语语境，东亚的「面子」和「脸」更强调关系性，不是普适的。给对方一个台阶，就是替他削减一次面子威胁。
- **④ 细颗粒操作**：在一次小摩擦里做一个「降级动作」——暂停一下、复述一遍对方的意思、给个台阶，然后观察走向。克己是不把小摩擦升级；敬是默认对方也在维护自己的面子。**降级是控制节奏、保留选项，和「当软柿子」是两种不同的行为**：坚定表达（说出自己的立场，但方式上不激化）与退让讨好（把立场压下去只求别起冲突）在心理学上是两类东西，不是一回事——这一条对曾被霸凌的读者尤其要写清，防止把「冲突降级」误读成「继续忍气吞声」。
- **⑤ 回到读者**：下一次小口角，试一个降级动作，看走向变了没有。本节的克己 / 克制与射_二「武德：在能力范围内保护他人」同源，但落在日常口角，不接权威建立。
- **本节红线提示**：戈特曼「90% 预测离婚准确率」直接剔除，不以任何形式出现；「情绪淹没」机制保留但证据改引独立神经科学；NVC 若用，只作「有一定合理性、但科学证据仍初步」的沟通脚手架，不说「研究证明有效」，可优先用罗杰斯反思式倾听；罗伯斯山洞实验若出现，只作历史图示并明确标注人为操纵与独立复现失败，核心论点「共同目标降低群体冲突」改用更独立的田野 / 元分析证据；标题与句子一律肯定式（「降级是控制节奏」而非「降级不是退让」）。

#### 子章节四：程序公平作为一种体验——被认真对待的感觉

- **① 真实案例**：同样一个对你不利的结果（分东西没分到、分工吃了亏、被拒绝），为什么这一次你没那么气。
- **② 打破直觉（数据自现）**：人对一件事的接受度，常常不取决于结果对自己是否有利，而取决于过程有没有让自己觉得被认真对待。分钱实验里，一方提议怎么分、另一方只能接受或拒绝（拒绝则两人都拿不到）——低于两成的报价约有一半会被拒绝，人宁可一分钱不拿也要惩罚不公平。另一个实验里，哪怕最终结果不变，只要让人把自己的看法说出来，他的不公平感就明显下降。
- **③ 权威框架**：「被听见」本身有重量——把话说完这个动作，独立于结果就能提升接受度。公平感是跨文化普遍存在的，人愿意为它牺牲自己的利益，只是具体的公平标准因文化而异（这一句取自对 15 个小型社会的实验，点到为止，不展开背后的宏观社会经济解释）。
- **④ 细颗粒操作**：在一次你说了算的小决定里，给相关的人一次「发声」机会（哪怕结果不变）、把理由说清楚、对每个人一个标准。这些都是成本极低的动作。
- **⑤ 回到读者**：本周在你经手的一件小事上，守住这几个动作，看对方的反应变了没有。
- **本节红线提示**：这一节严守「两三个人之间」的人际尺度，**全程避开「制度 / 规则 / 机制 / 治理 / 规范 / 原则 / 体系」这类词**——它们即便讲的是心理内容，形式本身也会暗示「这是一套治理指南」。泰勒的「四要素」不用编号清单 / 要点式呈现，拆散揉进具体人际小故事；组织公平元分析（机构语境粘性最强）不点名引用，改用「分钱实验 + 发声效应 + 密友冲突研究」这条更干净的人际证据链。

#### 子章节五：为什么有些群体能不靠外部强制维持合作

- **① 真实案例**：你待过的哪个「共用的东西」运转得好、哪个总出问题——宿舍的公共区、小组共享的一份笔记、健身房的公共器材。
- **② 打破直觉（数据自现）**：「公地悲剧」——共享资源必然被用光——出自 1968 年的一篇思想实验论说文，作者没去看过任何真实的牧场或渔场。后来研究者真的下地看了几十个案例，发现除了「国家来管」和「卖给私人」，还有第三种结局：用的人自己达成了默契，把资源维持了几百年。但这条路并不总走得通，很多共有资源确实也垮掉了。
- **③ 权威框架**：研究者回头看那些维持了很久的共用安排，发现它们碰巧都具备几个共同特征——大家分得清边界、付出和收益大致成比例、能看清谁在做什么、犯规按情节轻重递进处理而不是一刀切、有个低成本的地方能解决分歧。但这些特征并非同等靠得住：证据最强的是「盯着大家的人要对大家负责」，最弱的一条（多层嵌套的治理结构）只勉强到统计上「有点意思」的水平。连专门验证这些特征的学者本人都强调：这是**回头总结出来的共同点，不是照做就成的说明书**，别当「万能药」用——这句警告是本节最重要的护栏，必须写进正文。
- **④ 细颗粒操作（读者位置锁死在「参与者 / 观察者」）**：把这几个特征当一面「诊断镜」，看懂你正身处的一个小型共用安排为什么顺、为什么卡。共用的东西被人「顺走忘还」，往往由同伴当场轻轻提一句最有效（实验里，允许参与者自己对占便宜的人表达不满——哪怕自己要为此付代价——搭便车会明显减少），靠的是身处其中的人彼此愿意表达，而不是「有个专人在管」。但这种自己出面的纠正要**轻**：要有大家的共识撑着、不能升级成翻旧账或拉帮结伙的报复循环（约四分之一的惩罚会招来反报复，最后合作反而垮掉）；如果一个群体对「怎样算过分」本就没有共识，同样一句提醒可能被当成挑事，反而把秩序搞得更糟。
- **⑤ 回到读者**：作为一个共享安排里的参与者，你能看清它为什么运转、为什么失灵，并减少自己这一份摩擦。这一节不号召你去组建或加入任何持续存在的团体。
- **本节红线提示（全书政治红线最高的一节）**：祈使句一律改写成观察句（「研究者回头发现……具备哪些特征」，不是「你应该设计……」）；读者位置锁定「参与者 / 观察者」，不滑到「设计者 / 管理者」；八条特征必须点名最弱的一条 + 引用「别当蓝图 / 万能药」的护栏原话，不把八条讲成同等确定；境外案例（瑞士 Törbel 高山牧场、西班牙灌溉）只讲单一机制点、绝不作社会范本去羡慕效仿，西班牙案例必须引用历史学者对原始描述「过于和谐化」的修正；日本入会地案例搁置不用（除非团队单独裁定）；反社会惩罚的跨文化发现**只讲机制本身可能逆转，一律剔除具体国家 / 城市排名**（这是比治理红线更敏感的国别价值判断红线）；「92.1%」等未二次核实的数字改用「合作水平大幅提升」这类保守表述；八条特征不用清单式呈现，拆进人际小故事；公地悲剧双向平衡陈述，不单向讴歌自治。

### 3.3 全艺收尾（不升华、不布道）

挂在子章节五之后，冷静收束，不设升华段落：

> 危机能把陌生人临时拧成一股绳，因为生死把所有人的利害对齐了。日常没有这种外力，利害各自散着。礼这一套接口——稳住一条弱连接、一格一格建信任、给摩擦降速、让人觉得被认真对待、看懂一件共用的事为什么运转——就是在常态里，把摩擦降到不需要靠权威和暴力压着的水平。这里的每一样，你其实都已经在做一部分。哪一样你想多练一点，是你自己的事。

（写作纪律：「能力，不是义务」这层意思**嵌在语气里**——用「你可以」「是你自己的事」这种给选项的措辞承载，不写成「这是能力不是义务」这句口号，也不写「所以你应该」。）

## 五、文风红线清单（供后续「深写」阶段逐条核对）

定稿每一子章节前，逐条过一遍。这是本次重写的核心防线。

1. **多用肯定句**。陈述机制和事实，用正面表述。
2. **不用对比否定结构**。定稿前全文扫描「不是……而是」「并非……而是」「而不是」「既不是也不是」，逐条改写为肯定陈述。这是旧正文最严重的问题（28 处），也是本章头号红线——**节标题、开篇句、机制解释三处尤其高发**（旧版「降级不是退让」「互惠不是交易」「公地悲剧非必然」全部要改成肯定式）。
3. **节序上不用「先否定再纠正」的说教套路**。每节第二步（打破直觉）只摆案例和数字，让反直觉从事实里自己浮现；不写「你以为 A，其实是 B」这种先否定读者的判断再宣布正确答案的结构。结论由读者自得。
4. **少用比喻，绝不自造比喻**。优先事实陈述与逻辑推演。「承重结构」「进度条」「诊断镜」这类已在纲领 / 知识库里定型、朴素好懂的说法可用；不新造花哨比喻。
5. **不使用专业公式或学术化表述**。p 值、置信区间、条件概率记号、统计模型名、ratio 数值、学术框架原名一律留在 `knowledge/`；正文用白话 + 具体数字。
6. **书面语，不口语化**。语气克制，但不用网络腔、口水话、语气词堆砌。
7. **语气克制、冷静、精确，如侦探陈述调查结果**。摆证据、做推理，不替读者宣布答案。
8. **只摆事实、只做逻辑，不说教、不劝诫、不布道**。不写「你应该」「每个人都有责任」「让我们一起」。
9. **收尾不升华、不布道**。每节收在「回到读者处境 + 一个本周可做的具体动作」；全艺不设升华段落。「能力，不是义务」嵌进语气（给选项、用「你可以」），不写成口号。
10. **制造「我看我也行」的可达性**。每个操作都是低风险、可观察、不要求组织任何人的小动作；每个案例都是普通人做日常小事，不制造英雄。
11. **清单 / 编号呈现本身是风险信号**。程序公平「四要素」、Ostrom「八原则」这类学术框架，正文一律拆散揉进具体人际小故事，不用编号清单——清单形式本身会暗示「这是一套可执行的制度 / 体系」。
12. **证据分层、不把复现存疑的经典当既定结论**。戈特曼 90% 剔除、罗伯斯山洞降历史图示、NVC 标证据初步、Regan 可乐实验标无复现、邓巴数标统计争议、Axelrod 标两面——照第四节处理建议执行。
13. **结构变奏，防结构层的「下一句太好猜」**（现行《声音说明书》§0/§2/§7）。3.1 的五步流是**每节必须完成的五个功能，不是固定顺序模板**：开头在「真实场景 / 一句硬结论 / 直接发问」之间轮换，不许五个子章节全用场景开场；收尾都要落到「读者处境＋本周动作」这个功能上，但句式与形态逐节变化，不许五节都收在同一种「本周你可以试一次X」的句型上；小节长度、列举条数刻意不齐，不许每节都总-分-总。

## 六、政治边界与治理红线（承纲领第六节缓释规则）

1. **组织＝能力非归属**：全章不要求读者加入或组建任何持续实体；每个「共享 / 合作」案例都是读者已经身处的安排；衔接语气永远「因此你能做更多（你可以）」，不出现「所以你应该」。
2. **人际尺度锁死**：所有内容锁在「两个人或几个人之间如何降低摩擦」。出现「如何治理一个群体 / 如何建立规则约束一群人 / 如何设计一套制度」这类颗粒度，即为越界，收紧回人际界面。子章节五（Ostrom）与子章节四（程序公平）是最高危区，逐节复核。
3. **论证单元隔离**：中国本土痛点案例（宿舍公约、小组共享笔记、公共器材）只配中国本土合法解法，绝不与境外案例（瑞士、西班牙）放进同一论证单元。
4. **境外案例只作狭窄技术机制、不作社会范本**：瑞士 / 西班牙案例只取单一机制点，不用羡慕 / 效仿 / 借鉴措辞；日本入会地案例搁置（默认按「釜石先例」假设会被否，除非团队单独裁定）。
5. **国别价值判断红线**（本章特有、比治理红线更敏感）：反社会惩罚的跨文化研究，剔除一切按国家 / 城市 / 文化比较「合作规范强弱、法治强弱」的具体数据与排名，只讲机制本身可能逆转；不让读者读出「哪些国家 / 文化更擅长合作」。
6. **科普层级上限**：操作颗粒度的上限是「让读者觉得可信、愿意继续读」，不是「照做就能获得组织建设能力」。一旦发现自己在写「如何把一群人拉起来变成持续行动的组织」，退回到「这个机制怎么运作」的科普层级。

## 七、受众完整性与最不利端支线

### 7.1 常规人群修正

- **年龄分层**：主 16–24 岁尺度（兼职、合租、志愿、社团、线上社群、初入职场）；10–18 岁用同一机制换更小尺度的例子（同桌、小组作业、小区、班级公共事务），**只换例子尺度，不降机制深度**。
- **内向 / 外向**：弱联系按定义就是低频低强度，不预设读者外向；内向读者在此有天然优势。全艺不默认「会主动搭讪」为起点。
- **社交经验多寡**：信任「一格一格建」的逻辑本身就是给经验少者的保护性框架，不假设读者已有成熟社交技巧。

### 7.2 最不利端平行支线（不是低龄浅化，是并行的深度支线）

**最不利端组合读者**：极度社交焦虑 / 被孤立 / 曾被信任辜负或被霸凌。这条支线由 `knowledge/社交焦虑与孤立读者的零社交起点降级方案.md` 承重，贯穿子章节一、二、三。**跨艺首发权（2026-07-03 补）**：该知识库的六项核心证据（喜欢差距、聚光灯 46%/23%、一周陌生人对话 87%、孤独循环、干预元分析、渐进暴露）由**礼首发**——具体实验、数字在礼的正文完整呈现一次；乐复用同一证据链时一律回指＋增量（见乐大纲第五节表注），不把同一批数字在相邻两艺当新揭示各讲一遍。

- **零社交起点降级（子一）**：给出「不需要开口」的最低档——成为可靠的常在身影（重复曝光效应 [8]）、维护线上弱连接 [9]，而不是强制「主动多说一句话」。
- **反自责红线（贯穿）**：不默认「每个人都有一张社交网」，不把「连接少」写成个人失败。孤独是一个可被打断的认知—行为循环 [39]，不是性格缺陷。这条写成显性的语气纪律。对话后你被喜欢的程度、你的紧张被注意到的程度，都系统性地比你以为的轻 [36][37]；真实拒绝率远低于想象 [38]。
- **被辜负者的重构（子二）**：把「信任分层」显式讲成保护机制——小步、可逆的试探是理性，不是软弱 [15]。
- **降级不是退让的防误读（子三）**：坚定表达与讨好退让是两类行为 [22]，专门防止曾被霸凌读者把冲突降级读成「继续当软柿子」。
- **临床边界**：本书不承担心理治疗功能。若读者社交焦虑已达临床水平，正文明确建议寻求学校心理老师 / 专业帮助，不暗示「照练习做就能替代专业帮助」。
