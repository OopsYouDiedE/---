# 法规标准时效台账

> 依据：全库逐源核验（1424 条事实的时效字段）与定性调查。登记全部「依赖现行版本」的法规/标准/指南类引用——这类内容会静默过期（GB/T 7714 换代、治安管理处罚法修订都发生在本项目写作期内）。
> 用法：**某章进入渲染前，对照本表复核该章依赖的条目**（十几条，一次小检索即可）；每季度全表扫一遍作兜底。新知识入库涉及此类内容时，按 AGENTS.md §3 时效栏登记并同步进本表。
> 状态标记：✅ 现行已核｜⚠ 已换代/需动作｜⏳ 生效在即或版本在途。

## 四、复核规则

1. **章节渲染前**：按「引用文件」列反查该章依赖的条目，逐条确认状态仍为 ✅（一章通常十几条，小规模检索或人工即可）。
2. **季度兜底**：每季度全表扫一遍，重点是 ⏳ 与「约 X 年一版」类。
3. **两个已知待办**：交运规〔2025〕3 号补注（地铁手机外放）；NIMS 2017 核对（应急后勤）。
4. 新增知识涉及法规/标准/现行指南时，入库的同时在本表登记一行。
