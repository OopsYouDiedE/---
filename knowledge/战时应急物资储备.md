# 事实核查：战时/灾难应急物资储备

**核查日期：** 2026年6月25日  
**核查用途：** 新六艺·「射」章·「军事基础能力」模块事实支撑  
**信源类型：** 瑞典、芬兰、瑞士、美国、台湾政府官方指南；WHO/SPHERE人道主义标准；FEMA应急管理文献

## 一、各国官方储备标准对比

### 1.1 瑞典《若危机或战争来临》（*Om krisen eller kriget kommer*）

- **✓ 官方文件证实**：2018年，瑞典民事应急局（MSB）将该手册发放至全国490万个家庭邮箱，2022年更新版增加了战时条款，2024年国防法案将建议储备期延长至**3个月**。
- **最低储备建议（2022版）**：
  - **水**：每人每天至少3升，储备至少1周
  - **食物**：1周以上的无需冷藏食品
  - **能源**：蜡烛/手摇发电灯、便携煤气炉
  - **通讯**：手摇或电池收音机（接收SR Sveriges Radio官方紧急广播）
  - **现金**：小额纸币（500–1000克朗等值）
  - **药品**：常用药至少1周用量，处方药至少1个月
- **战时附加内容**：了解最近的防空洞/避难所位置；如收到疏散命令，带好证件、药品、充电设备和现金在30分钟内出发；不传播未经核实信息。
- **信源**：MSB官方平台（msb.se）；*Om krisen eller kriget kommer* 2022版。

### 1.2 芬兰（扩展版：战时场景）

- **✓ 官方数据证实**：芬兰红十字会和国家应急供应局（NESA/Huoltovarmuuskeskus）联合建议72小时家庭自给，但俄乌战争后，NESA明确将战时推荐储备期调整为**2周**。
- **战时重点增补**：
  - 收音机（FM波段，接收Yle广播）列为最高优先
  - 碘片（KI片）——用于核事故或辐射威胁；芬兰政府已向全国家庭免费分发
  - 建议每家庭储备约**90升**饮用水（即每人每天3升×2周×家庭人数）
- **信源**：芬兰应急供应局官网（huoltovarmuuskeskus.fi）；72tuntia.fi官方平台。

### 1.3 瑞士联邦经济供应局（FONES/BWL）

- **✓ 官方制度证实**：瑞士实行双轨储备体系：
  - **企业强制储备（Pflichtlager）**：法律规定粮食、燃料、药品等关键物资企业必须储备3至6个月供应量，由联邦政府统一调度。
  - **个人建议储备**：BWL向全体居民发布《个人应急储备》指南，建议储备**2周**的日常用品，内容包括：
    - 饮用水：每人每天2升×14天
    - 卡路里密集食品：米、面、罐头、干豆、能量棒
    - 卫生及急救物资
- **信源**：瑞士联邦经济事务部官网（bwl.admin.ch）。

### 1.4 美国FEMA建议

- **✓ 官方标准证实**：FEMA"Ready.gov"计划区分两种储备规格：
  - **72小时便携应急包**（"Go Bag"）：随时可带走，用于强制疏散
  - **居家2周延伸储备**：用于就地避难（Shelter-in-Place）
- **水标准**：每人每天**1加仑（3.78升）**，包含饮用和基本卫生用水；炎热气候或体力劳动条件下加倍。
- **信源**：FEMA官方平台（ready.gov）；美国疾控中心（CDC）应急准备指南。

### 1.5 台湾

- **✓ 官方政策证实**：台湾全民防卫动员准备法授权政府在紧急状态下征用物资。台湾内政部消防署建议家庭储备**72小时**应急物资，包括饮水（每人每日2升）、食品、药品、重要文件复印件、现金及手机充电宝。2024年"汉光"演习后，部分县市建议将标准提升至**1周**。
- **信源**：台湾消防署官网（nfa.gov.tw）；内政部防灾资讯服务网（disasterp.gov.tw）。

## 二、水的储备

### 2.1 人均需求标准

| 场景 | 每人每日最低需水量 | 来源 |
|---|---|---|
| 饮用（基础生存） | 2.0 升 | WHO最低标准 |
| 饮用+基本卫生（如厕/简单擦洗） | 3.5–4.0 升 | FEMA / 瑞典MSB |
| 饮用+卫生+简单烹饪 | 5–7 升 | SPHERE人道主义标准 |
| 充分卫生（含淋浴） | 15 升 | SPHERE完整标准 |

**实操建议**：以**每人每天5升**为基础储备计算，2周家庭储备量=5升×人数×14天。4口之家需储约280升。

### 2.2 储水容器与保质期

- **食品级PE容器**：避免使用矿泉水以外的普通塑料桶（有渗出化学物质风险）。市售食品级储水桶（20升、200升）可购于户外装备店。
- **保质期**：自来水灌入密封容器，常温避光存放，建议**每6个月更换一次**（即使无异味）。商业瓶装水可储至标注保质期。
- **大型储备**：可利用浴缸应急储水（标准浴缸容量约200–300升），紧急断水前快速灌满；专用"WaterBOB"浴缸储水袋可隔离污染，储水110升，约国内网购可得。

### 2.3 净水方法对比

| 方法 | 适用场景 | 有效对象 | 注意事项 |
|---|---|---|---|
| **煮沸（1分钟滚沸，高海拔3分钟）** | 有燃料时首选 | 杀灭细菌、病毒、寄生虫 | 不去除化学污染物 |
| **含氯消毒片/漂白粉** | 无燃料、大量水源 | 细菌、病毒（对隐孢子虫效果弱） | 静置30分钟后使用；每升水约2片 |
| **过滤+消毒组合（如Sawyer/LifeStraw滤水器）** | 野外/长期使用 | 过滤细菌和寄生虫（需配合消毒去除病毒） | 对化学污染和重金属效果有限 |
| **商业RO/超滤净水器** | 家庭日常 | 细菌、病毒、重金属、氯 | 需要电力；滤芯需储备 |

## 三、食物储备

### 3.1 热量需求基准

| 人群 | 静息状态 | 轻度活动 | 中度体力劳动/应激状态 |
|---|---|---|---|
| 成年男性（70kg） | 1700 kcal | 2200–2500 kcal | 3000–3500 kcal |
| 成年女性（60kg） | 1400 kcal | 1800–2000 kcal | 2400–2800 kcal |
| 儿童（6–12岁） | 1200 kcal | 1400–1600 kcal | — |
| 老人（65岁以上） | 1600 kcal | 1800 kcal | — |

### 3.2 长保质期食物选择（适合中国家庭）

| 品类 | 保质期参考 | 热量密度 | 烹饪需求 |
|---|---|---|---|
| 真空包装大米/糯米 | 3–5年 | 360 kcal/100g干重 | 需水+燃料 |
| 精盐 | 无限期（密封） | — | 无 |
| 白砂糖/红糖 | 无限期（密封干燥） | 400 kcal/100g | 无 |
| 蜂蜜 | 实际无限期 | 300 kcal/100g | 无 |
| 压缩饼干（军用型） | 3–5年 | 450–500 kcal/100g | 无 |
| 速食方便面（低水分干燥版） | 1–2年 | 440 kcal/100g | 最低（可干食） |
| 铁罐头（午餐肉/沙丁鱼/豆豉鲮鱼） | 3–5年 | 200–350 kcal/100g | 无（开罐即食） |
| 脱水蔬菜包 | 2–3年 | — | 加水泡发 |
| 花生/核桃/杏仁（真空密封） | 1–2年 | 550–650 kcal/100g | 无 |
| 奶粉（全脂密封） | 1–2年 | 500 kcal/100g | 加水 |
| 军用/户外冻干口粮 | 10–25年 | 400–600 kcal/包 | 加热水 |

**优先原则**：无需烹饪>少量加水>需烹饪。紧急状态下燃料和水都是稀缺资源。

### 3.3 特殊人群需求

- **婴幼儿（0–1岁）**：储备足量配方奶粉（按当前用量×储备天数），及消毒奶具
- **糖尿病患者**：低糖食品优先；严格备足胰岛素（冰箱失电后胰岛素有效期约28天，室温25°C以下）
- **哺乳期**：热量需求增加500 kcal/天；额外储水
- **老人**：软质食物优先（稀饭、软罐头）；注意慢性病用药连续性

### 3.4 储存原则

- **温度**：≤25°C，避免温度剧烈波动；地下室/北面内墙储藏室最优
- **避光**：紫外线加速食物劣化，避免透明容器
- **防潮**：密封容器+干燥剂（硅胶或石灰）
- **防虫防鼠**：密封金属/厚壁PP容器；避免纸箱直接存储
- **先进先出（FIFO）**：每次补货放在后，取用从前拿，定期（每6个月）清点并消耗临期品

## 四、药品与医疗物资

### 4.1 基础OTC药品清单

| 类别 | 推荐药品（通用名） | 用途 |
|---|---|---|
| 止痛/退热 | 对乙酰氨基酚（扑热息痛）、布洛芬 | 发热、头痛、肌肉痛 |
| 止泻 | 洛哌丁胺（易蒙停）；口服补液盐（ORS） | 急性腹泻及脱水 |
| 抗过敏 | 氯苯那敏（扑尔敏）、西替利嗪 | 过敏反应、花粉症 |
| 消化 | 铝碳酸镁（胃达喜）、蒙脱石散 | 胃酸/腹泻 |
| 外用消毒 | 碘伏、医用酒精（75%）、生理盐水冲洗液 | 伤口处理 |
| 烫伤 | 烫伤膏（京万红等）或美宝湿润烧伤膏 | 轻度烫伤 |
| 眼部 | 氯霉素眼药水 | 结膜炎 |

### 4.2 包扎/止血物资清单

- 医用绷带（弹力绷带×4卷；纱布绷带×4卷）
- 医用胶布/创可贴（大中小各一盒）
- 无菌纱布块（10×10cm×20片）
- 止血带（CAT止血带或等效品×2）——对于大出血四肢伤口，这是唯一有效手段
- 创伤止血敷料（壳聚糖/高岭土止血纱布，如QuikClot或国产等效品）
- 医用镊子、剪刀各1把
- 一次性无菌手套×20副
- 急救毯（铝箔保温毯）×2

### 4.3 处方药储备策略

- **慢性病用药（高血压/糖尿病/心脏病等）**：提前与医生沟通，尽量储备**3个月用量**；中国医保规定部分慢性病可一次开具3个月长期处方
- **胰岛素**：需冷藏（2–8°C），断电后开封瓶在≤25°C室温下保质28天；建议储备一台小型车载冰箱或蓄冷箱
- **碘化钾（KI）**：核事故专用，防止甲状腺吸收放射性碘；需在核事故发生前服用方有效；成人130mg/次，儿童65mg；普通家庭可向医疗机构咨询获取

### 4.4 辅助器具备份

- 眼镜：储备一副旧镜（与当前度数接近即可）
- 隐形眼镜：多备3个月存量；断电/断水条件下优先切换回框架眼镜
- 助听器：备用电池至少3个月用量
- 轮椅/助行器：核查零配件，确保备有充气泵/备用轮胎

## 五、能源与通讯

### 5.1 收音机（最关键的通讯工具）

- **✓ 多国官方一致确认**：在断电/断网情况下，调幅/调频（AM/FM）收音机是接收政府应急广播的唯一可靠工具
- 中国官方应急广播：中央人民广播电台（AM 639 kHz；FM因地区而异）；各省市应急广播频道
- **推荐配置**：手摇+太阳能+电池三用收音机（约100–300元，国内主要网购平台均有售）；备用5号/7号碱性电池各8节

### 5.2 备用电源

| 设备 | 容量参考 | 适用场景 | 注意事项 |
|---|---|---|---|
| 大容量充电宝（30,000–60,000 mAh） | 手机充电5–10次 | 短期（1–3天）通讯维持 | 每3个月充放电一次维护电量 |
| 便携太阳能板（100W） | 晴天可对充电宝/12V蓄电池充电 | 中期（1周+）自给 | 效率受天气影响大 |
| 铅酸/磷酸铁锂蓄电池+逆变器 | 100Ah蓄电池≈1.2 kWh | 较大功率设备（收音机、应急照明、医疗设备） | 放置于通风处，铅酸电池防止过充 |
| 便携发电机（汽油） | 500–2000W | 短期大功率需求 | 必须室外使用，严禁室内（CO中毒） |

### 5.3 照明

- LED头灯×家庭成员人数（1人1个，备足电池）
- 应急荧光棒（化学发光棒）×10支（无需电力，持续约8小时）
- 蜡烛×20支+金属烛台（防火意识：不可无人看管）

### 5.5 家庭成员追踪服务（ICRC RFL）

- **✓ 国际红十字委员会官方项目**：国际红十字委员会（ICRC）在武装冲突与重大灾害期间运营**Restoring Family Links（RFL，恢复家庭联系）**服务，专门协助在战乱或灾难中失散的家庭成员重新取得联系。
- **适用情境**：冲突爆发后家庭成员分离、撤离路径不同导致失联、拘留或伤亡信息不明时。
- **联系方式**：通过当地红十字会/红新月会分支机构提交寻人请求，或访问 ICRC 官网（icrc.org/family-links）。中国联系渠道：中国红十字会总会（redcross.org.cn），有相应家庭联系协作机制。
- **服务范围**：战时通讯断绝后，RFL 通过各国红十字网络传递"生死存亡"信息（是否存活、所在地点），而非完整通讯，但这是最关键的信息。
- **⚠ 注意**：此服务优先处理武装冲突情境；普通灾害（地震、洪水）通常由民政部门及当地应急管理部门的人员寻找系统优先承接，ICRC 作为补充。
- **信源**：ICRC 官网（icrc.org）；中国红十字会总会（redcross.org.cn）；ICRC Annual Report 2023 *Restoring Family Links* 章节。

### 5.4 烹饪与加热（室内安全规则）

| 设备 | 室内使用 | 注意事项 |
|---|---|---|
| 卡式炉（丁烷气罐） | **谨慎允许**：必须开窗通风 | 单罐燃烧约1小时；储备10–15罐；气罐避免高温存放 |
| 固体酒精（固体燃料块） | 可用，需通风 | 热值低，适合加热不适合长时间烹饪 |
| 汽油炉/煤油炉 | **不建议室内使用** | CO和气味问题；适合阳台/室外 |
| 电磁炉/电热锅 | 有电时首选 | 高效安全；无电时无用 |

> **严格禁止室内使用**：炭炉、木柴炉、汽油发电机——每年均有大量一氧化碳（CO）中毒死亡事故来自此类误用。

## 六、重要文件

### 6.1 需备份的文件清单

| 文件类型 | 备份方式 |
|---|---|
| 身份证、户口本、护照 | 彩色复印件×3份；扫描PDF |
| 房产证/租房合同 | 扫描存档 |
| 银行卡号码/账户信息 | 手写记录（不存手机）；密文备忘 |
| 医疗记录/处方/病历本 | 扫描PDF；重点页复印 |
| 保险单（寿险/医疗险/财产险） | 扫描存档+记录紧急联系电话 |
| 家庭成员紧急联系人电话 | 手写纸质版（手机没电时可用）|

### 6.2 防水保存

- 塑封（覆膜）处理后放入密封防水袋（如Ziplock双层密封袋）
- 重要文件放入**A4防水文件袋**（市面约30–100元）
- 存放于阴凉高处（防水灾）

### 6.3 数字备份策略

- 将所有文件扫描为PDF，存入加密U盘（使用VeraCrypt等工具）
- 同步至私有NAS或加密云端（不建议直接上传未加密证件至公共云）
- 每半年更新一次备份内容

## 七、现金

- **✓ 各国民防指南一致建议**：灾难初期（3–7天）ATM/银行可能停运、电子支付中断，现金是唯一可用的支付手段
- **建议金额**：至少相当于家庭**2周基本生活开支**的现金（因家庭情况而异，一般建议2,000–5,000元人民币）
- **面额分配**：大量小额（1元、5元、10元）优于大面额；零钱在混乱时期找零困难
- **存放原则**：分散存放（家中多处+随身携带应急包中少量）；不可全部放在一处

### 8.2 国内可购置的关键装备

- **手摇/太阳能收音机**：德生（Tecsun）、熊猫等品牌，80–200元，京东/天猫均有售
- **止血带（CAT类）**：淘宝搜"战术止血带"，国产版本50–100元；建议选橙色（符合国际急救标准颜色识别）
- **应急净水器**：多布（LifeStraw国内版/Sawyer国内授权版）或国产类似品，150–500元
- **防水储水桶（20L食品级）**：超市/五金店，约30–50元/个
- **应急毯（铝箔保温毯）**：登山用品店，约5–20元/张
- **多功能卡式炉**：岩谷（Iwatani）或国产品牌，100–300元；配套丁烷气罐可网购
