## 一、残疾人总数："8500万"是2010年的比例外推值

[已多源确认——中国残联官网一手文本]

- **2006年第二次全国残疾人抽样调查**（标准时点2006年4月1日）：推算全国残疾人总数 **8296万人，占总人口6.34%**。这是最近一次实测调查。
  - 来源：中国残联《第二次全国残疾人抽样调查主要数据公报（第一号）》: https://www.cdpf.org.cn/zwgk/zccx/dcsj/9ff8c67574af479dabaf6ac21b5305f4.htm 可信度：★★★★★
- **"8500万"的唯一官方出处**：中国残联《2010年末全国残疾人总数及各类、不同残疾等级人数》——用2006年调查的6.34%比例乘以第六次人口普查（2010）总人口外推，得**2010年末8502万人**，此后各界沿用至今。**不是新调查**。
  - 来源：https://www.cdpf.org.cn/zwgk/zccx/cjrgk/15e9ac67d7124f3fb4a23b7e2ac739aa.htm 可信度：★★★★★（原文已抓取确认）
- 数据滞后是官方承认的现实：2024年有全国人大代表建议尽快启动第三次全国残疾人抽样调查；2021年起残联改做"全国持证残疾人基本状况调查"（仅覆盖持证人群）。截至2026年7月未见第三次抽样调查启动。（网易转述 https://www.163.com/dy/article/ISK7DJ1B0512D03F.html ★★★☆☆）
- **写作口径**：写"官方常用口径约8500万，出自2010年比例外推；最近一次实测是2006年"。直观换算：约每16个人里有1个（6.34%）——这是"少数群体其实不少"的最硬论据。

## 二、各类别残疾人数

[已多源确认——cdpf.org.cn 一手数据]

2006年第二次抽样调查 / 2010年末推算（万人）：

| 类别 | 2006（万人 / 占比） | 2010年末推算（万人） |
|---|---|---|
| 视力残疾 | 1233 / 14.86% | 1263 |
| 听力残疾 | 2004 / 24.16% | 2054 |
| 言语残疾 | 127 / 1.53% | 130 |
| 肢体残疾 | 2412 / 29.07% | 2472 |
| 智力残疾 | 554 / 6.68% | 568 |
| 精神残疾 | 614 / 7.40% | 629 |
| 多重残疾 | 1352 / 16.30% | 1386 |

- 2010年推算按等级：重度2518万，中轻度5984万。
- 章内可用的量级感：**肢体残疾最多（约2400万）**，听力残疾第二（约2000万），视力残疾约1200万——每一类都是千万量级。

## 三、持证残疾人数（注意口径变化）

[官方一手数字止于2018；之后公报不再公布总量]

- **2018年末：3566.2万人**（《2018年残疾人事业发展统计公报》原文："全国残疾人人口基础数据库持证残疾人3566.2万人"）。
  - 来源：https://www.cdpf.org.cn/zwgk/zccx/tjgb/2e16449ca12d4dec80f07e817e1e3d33.htm 可信度：★★★★★
- 2022年末3768.166万——仅见光明网转述（https://topics.gmw.cn/2024-05/17/content_37329070.htm ★★★☆☆），残联对应一手页面未找到。
- 残联2022/2023/2024年度统计公报**不再公布持证总数**，只公布分领域数据（如2024年公报：持证残疾人就业914.4万人；接受基本康复服务866.8万人次 https://www.cdpf.org.cn/zwgk/zccx/tjgb/1706f34657364af9a52b67f77d8c9f2b.pdf ★★★★★——原链接为.htm，此处保留检索所得页面）。
- **写作口径**：持证约3600万（2018年口径）vs 推算总数8500万——**过半残疾人没有办证**，"你身边看不见≠不存在"的又一论据。引"3600万"必须注明2018年。

## 四、视力障碍与听力障碍的具体数字

[口径提示：官方"视力残疾"分级概念≠媒体"盲人/视障"概念，不可混用]

- **视力残疾**：2006年单项1233万；**含多重残疾中的视力残疾共1691万**（中国盲人协会官网，源自2006年调查 http://www.zgmx.org.cn/newsdetail/d-13365-0.html ★★★★☆）。按等级：一级（最重）占24.2%。
- **"全盲"人数：官方从未发布口径统一的数字**。各二手估计（WHO 2010约824.8万失明；"2020年初步统计"约830万；盲协原副主席2021年称视障含低视力1731万）差异大，均★★★☆☆——正文如需，写"视力残疾逾千万，其中全盲的估计在数百万量级"，不咬单一数。
- **听力残疾**：2006年调查2004万人（24.16%），2010年末推算2054万（cdpf.org.cn 一手 ★★★★★）。无更新的全国性调查。

## 五、导盲犬数量

[数字口径混乱——正文用"累计交付数百只"量级]

- **中国导盲犬大连培训基地**（2006年成立，大连医科大学内，中国大陆首家）：截至2024年12月**累计培训并免费交付391只**，服役于28个省市。
  - 来源：央广网 2024-12-18: https://dl.cnr.cn/jdt/20241218/t20241218_527012810.shtml 可信度：★★★★☆
- 2021年时点：大连基地累计毕业239只，年产约35只，淘汰率约55%；全国共5家训练基地（大连、上海、广州、西安、郑州）。
  - 来源：广东残联转载: http://www.gddpf.org.cn/xwzx/mtgz/content/post_1052900.html 可信度：★★★☆☆
- **"全国在役总数"无可靠官方数字**（累计培训≠在役，媒体口径200–410只之间波动，另见 knowledge/视障引导手法与导盲犬礼仪.md 第2.4节）。
- **最硬的对比写法**：视力残疾约1200万–1700万人 vs 累计交付导盲犬约400只量级——"每几万名视障者摊不到一只"。粗略换算说法（"每9万视障者等1只"）为媒体演算 ★★★☆☆，正文用自算量级即可，不引这句。

## 六、写作提醒（检索发现的三个坑）

1. **"8500万"要写清是2010年比例外推**，非实测；最近实测是2006年。
2. **"3600万持证"对应2018年**，此后公报不再公布总量。
3. **"全盲人数""在役导盲犬总数"均无统一官方数字**——用替代口径（视力残疾分级数 / 累计交付数）并标明。
