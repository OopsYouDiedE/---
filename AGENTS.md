## 1.本书设计理念

本书主要针对16-24岁青年人进行书写，其内容以受过普通高中教育的大学生为基准。如果内容在不同人群中存在不同适用情况，则本书将进行补充说明，而不是删掉内容。本书通过使用职业流程和标准规范进行指导，来确保本书的深度足够日常使用。

在本书设计中，我们将围绕六艺的本意，通过现代化改造，来试图建立一个在新时代能够适用的培养手册。我们鼓励读者阅读完本书后，将介绍的专业工作流程用于日常生活处理问题中。同时，这也是给设计者本人的一本生活参考书，以实用和可靠作为标准。

## 2. 工作流要求

对正文内容进行撰写，补充，修改（除语气修改外），均需要遵循下面流程。
首先，对待编写或者补充内容，进行知识库搜集。知识库更新遵守更新规范。在进行完毕后，才能编写正文内容。

## 3. 知识库更新规范

凡是涉及到事实，规范，指南和建议时，优先使用权威官方给出的内容，禁止以自己的知识库作为论据。
如果是因为拒绝访问而不是互联网上不存在导致无法查明，必须进行标注，而不是假装某件事不存在。
对于一件涉及较新内容的事。请你务必不要相信你的知识库的判断，而是优先上网进行查证。
对信源和单条事实可靠性，你需要进行如下分类。
| 信源分类 | 标准 | 举例 |
| --- | --- | --- |
| A | 在该话题下，该机构被公认为权威和专业，能够不受利益干扰的撰写内容，历史可靠。 | 顶级学术期刊，国家权威机构发布的指南，正统行业协会发布的规范 |
| B | 该机构被公认为权威或者专业，但在该话题下，有时会存在轻微问题 | 国际主流媒体，主流学术期刊，各家机构给出的调查报告 |
| C | 对其权威性或者专业性有疑问，但过去有提供有效信息的记录 | 媒体评论，专业人士建议，名人发言 |
| D | 对其有重大疑问，过去的记录时好时坏 | 一般媒体评论，社交发帖等 |
| E | 有编造内容或者提供无效信息的历史，或者内容无法溯源、已被证伪 | 不可信官方媒体，有造假历史的机构，无法溯源的消息 |
| F | 没有过往记录，无从判断。F 不等于不可信，全新来源默认为 F | 第一次接触、没有历史记录的机构或者个体 |

字母评级只看历史记录，不看知名度。信源在该话题下存在利益关系的（如企业公布自家产品的数据），不降字母，在条目中标注「利益相关」并从严交叉证实。

| 内容可信度分类 | 标准 |
| --- | --- |
| 1 | 具有确凿原始出处，被多个相互独立的可信来源交叉证实，逻辑自洽 |
| 2 | 具有确凿原始出处，逻辑自洽，与已知信息一致，但缺少独立证实 |
| 3 | 确凿原始出处无法考证，但可以被多个来源交叉印证 |
| 4 | 只有单一可信信源可以证实，或者暂无其他信息可比 |
| 5 | 被更可信的信息反驳，已被证伪，或者与常识不符且没有凭据 |

无法判断的全新内容按 5 保守处理，但标注「无从判断」而不是「假」。时效性不参与 1–5 评分：内容是否为现行版本、被什么取代，记在条目的时效栏。

使用门槛：可信度 1–2 可进正文；3–4 可进正文但必须注明局限；5 以及 E、F 类信源不得作为正文论据，只可标注后作背景参照（F 补齐记录后重评）。交叉证实要求来源相互独立，同一上游的转述只算一个来源。

入选知识库每条具体知识的格式包括：
```markdown
搜索时间 | 搜索内容 | 评级（信源字母+内容数字） | 时效（更新于/失效条件） |
---
知识内容
---
来源列表和原文摘引
```


## 4. 文风与语气规范

DO：

- 用肯定句。
- 用书面语。
- 使用事实和逻辑论证。
- 标题简单直接，能够看懂本章内容。
- 让读者通过我们给的证据和事实在内心得出结论。
- 在应该由作者阐述思想的地方，由作者本人阐述。

DONT UNLESS HAVE TO：

- 进行说教。
- 洗脑，布道，灌输答案。
- 给小孩子的语言风格。
- 用复杂从句。
- 自己制造比喻性词汇。
- 使用对比否定结构。
- 政治体制的比较性暗示，偏见，政治动员。

LIMIT：

- 专业公式和行业黑话。
- 修辞，夸张，字母缩写，语气词。

## 5. 并行任务与模型分配

互不依赖、不需要共享中间结果的子任务，拆成并行 SubAgent 执行，提升吞吐。
内容概括、单条事实核对比较、检查文章标注引用是否正确、机械性重复劳动： Haiku。
内容溯源、知识库写入等： Sonnet或Opus。
编写内容，监督SubAgent运行，分析调查方向：Opus或者是Fable。

## 6. 政治边界

- 避免任何可被解读为推翻政权或策划组织性运动的措辞。
- 外国军事动员、对抗类案例先过落点测试：叙事落点是平民疏散、自救互救或民防，可用；是国家间军事对峙或政权动员，不用。
- 硬底线：军事与政治领域不用日本和台湾的案例，自然灾害与应急救援除外；与落点测试二者取严。
- knowledge/ 条目涉及受限案例时，就地标注「仅供背景参照，不得进入正文」，不写「建议正文采用」类提示。
- 引用中国官方数据与官方口径时，客观引述。
