# 新六艺 · 写作规范与项目规则

## 项目定位

本书是一本面向 10-18 岁青少年的现代公民自我培养指南，以君子六艺为文化引子，以实践为驱动，强调组织能力、协同能力与独立思考。

核心要素：培养一个具有组织和团结能力，能够对现代社会具备基本理解的公民。

## 目标读者画像

本书的付费读者和潜在支持者具有以下特征：
- 对原子化社会中人与人之间的疏离和脆弱感到担忧
- 关注军事与政治议题
- 对当前中国教育体系存在明确不满
- 有行动意愿，但缺少组织框架和同类群体

书中内容应针对这一群体的核心痛点进行设计。

## 内容准确性要求

所有涉及专业领域的内容（体能训练、急救、灾害应急、军事素养等），必须基于官方教程、权威机构发布的实践指南或有据可查的实践记录。禁止凭印象编造细节。

### 知识管理规范

通过网络检索采集到的事实性内容，统一记录至项目根目录下的 `knowledge/` 文件夹中。具体要求：
- 每条采集内容以其主题作为文件名，使用 Markdown 格式存储。
- 记录原始信源的名称与链接。
- 每条事实应使用至少两个可信信源进行交叉证实，并在条目中标注各信源。
- 未经交叉证实的内容应标注「待证实」，在正式写作中不得直接引用。

## 文风与语气规范

### 句式
- 多用肯定句式，少用否定句式。
- 少用复杂从句，保持句子结构简洁直接。
- 避免频繁使用「不是……而是……」「并非……而是……」等对比否定结构。
- 使用书面语，拒绝口语化表达。

### 修辞
- 少用比喻，优先使用事实陈述和逻辑推演。
- 不使用专业公式或学术化的表述方式。
- 行文应当制造「我看我也行」的参与感和可达性，降低读者的心理门槛。

### 态度
- 只摆事实，只做逻辑论证。不说教，不劝诫，不布道。
- 结论由读者自行得出。作者的角色是呈现证据和推理，而非宣布答案。
- 整体语气应当克制、冷静、精确，如同侦探在陈述调查结果。

## 政治边界

- 本书的定位是现代公民的自我培养指南。所有内容服务于这一定位。
- 避免任何可被解读为推翻政权或策划组织性运动的措辞。
- 引用外国案例时，除自然灾害与应急救援领域外，不得使用日本和台湾的案例。
- 引用中国政府官方数据和官方承认的逻辑（如「黄金72小时」自救互救概念）作为论证基础时，以客观引述的方式呈现。
