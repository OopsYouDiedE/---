## 零、一句话定位与魂（沿用 v3，收紧）

**御 ＝ 移动的德行：当你移动、以及组织人和物移动时，如何既到得了，又不把危险甩给别人；如何可靠地把人和东西带到该去的地方。**

- **魂**：射的魂是「力量用于守护」；御的魂是它的姊妹——「**移动用于守护**」。**御_二「移动的德行」是这个魂的道德中心**，向下渗进每一章。
- **四章足矣**。射第五章那种"危机建权威"的特殊位，御没有，不凑第五章。
- **practiceability 不是门槛**：教会知识、能练的练（含模拟器/游戏）、剩下的当准备。唯一不被撤销的是**律7 政治信号**——御全程民用，干净。
- **声部**：身体（机动），偏操作，T≈40–43%（与射同档）。

现盘四章逐句读是通的、深的、诚实的、在嗓音上的。重写不为救差稿，为把**架构、深度、压轴**三件事一次做到位：

2. **深度整体上抬到"较为专业的模拟游戏也能用得上"。** 每章给一个具体的**深度靶**和对应的游戏品类（见第四节）：一个玩拟真摩托模拟、铁路调度模拟、空管模拟、工厂产线经营的读者，读完这一章，应当能把书里的机制**认出来、用得上**。深度往这个坐标顶，而不是停在"注意安全"的科普层。

3. **御_四 升为全御实践高峰。** 它零资源、零权威、当下可做、弱者可达性最高，又最古典（御者管辎重）。让它成为压轴：把前三章关于"流"的所有理解，最后落进读者自己的两只手（见第五节）。

**外加一条贯穿全御的深度脊——「流的物理」**：
`御_一 的跟车间隔（人的尺度）→ 御_三 的发车间隔＋排队论（系统的尺度）→ 御_四 的人力流水线（把空手往返删掉的流）`。
同一条"流怎么才不堵、不撞、不空转"的物理，在三个尺度上重演。这条脊既让四章咬合成一部，也正是"模拟游戏可用"那种深度的统一骨架。深写时**显式回扣**这条脊（一 埋、三 接、四 收）。

## 二、四章结构与叙述弧

叙述顺序保持 `一→二→三→四`，因为它是一条干净的**能动性放大弧**，且终点正好落在实践高峰：

```

御_三 组织物流运输     → 你怎么组织人和物大规模地动   （系统尺度：调度，模拟游戏深度中心）
```

**重点（emphasis）的重新分配**——这是 v4 相对 v3 最大的调整：

## 三、御_一 / 御_二 分工边界（硬约束 · 杀重叠）

> **v5 重定位（2026-07-02）**：作者裁定 御_二"完全错误"，视角主体从"弱者的自保翻面"上移为"**握上方向盘、尤其大车方向盘之后的德行**"。一→二 由"镜像"升级为"**弱者 → 强者（司机／大车司机）**"的进阶。（作者离开期间按最强解释拟定，待确认；若改 opt2"只写司机德行、骑车行人移出"或 opt3"维持现状加重"，本节相应调整。）

**总规则一句话：**
- **御_一 ＝ "你还是弱者时，怎么保住自己"**——包括把自己弄得**被看见**。
- **御_二 ＝ "你握上杀器（汽车／大车）之后的德行"**——不撞到别人、给后车预警前方风险、克制、同时护住自己。**主体是司机（尤其大车司机），不再是"路上的小角色"；骑车/行人德行降为收尾一小节。**

**进阶装置（弱者→强者）**：同一套"看见会两头失灵"的物理，一 从"谁看不见我→我得让自己被看见（弱者自保）"这头讲；二 从"**我坐进驾驶位后，我看不见谁→那些人就是我的责任**（强者清死角）"那头讲。凡在一 讲透的物理/机制，二 只引用、只把视角换到方向盘后，绝不重讲步骤。

## 四、深度靶：每章"模拟游戏可用"的具体标准（本次重点）

给每章一个**深度坐标**：想象一个玩对应品类**较为专业的模拟游戏**的读者，这一章要深到他能把机制认出来、用得上。下面列"现稿已到哪 / 本次要加到哪 / 对应游戏品类 / 需补库"。**游戏品类仅用于给深度定靶；正文严守"只说通用品类、绝不点名具体游戏"的红线（见第七节游戏纪律）。**

### 御_一（对应：拟真摩托 / 第一人称拟真驾驶模拟）
- **现稿**：KE=½mv²、两段停车、ATGATT 各件降险数、SEE、Hurt/MAIDS、countersteering、前后 70/30 制动、ABS、3-4s 跟车、致命失误。
- **本次加到**（"读者能看懂机器在干什么"）：
  - **抓地预算 / 摩擦圆**：轮胎的抓地力是一份预算，刹车和转弯**共用**这一份——为什么"入弯前减够速、弯中别再猛刹"是物理必然，不是口诀。
  - **反向转向为什么成立**：陀螺进动 + 几何两条解释各给一层，落到"高速避险是推不是拉"的下意识动作。
  - **制动时的重量转移**：为什么前轮吃大头（载荷前移、前轮抓地上升、后轮易抱死），threshold braking 与 ABS 在这条曲线上各自解决什么。
  - **接触斑块 / 循迹**：轮胎与地面就巴掌大一块，决定了一切；湿滑/砂石为何让预算瞬间缩水。
  - 锁死在**安全基线**，不下探赛道 trail-braking 竞技技巧。
- **补库**：`载具与道路安全.md`（125k）已厚，需**补摩托/车辆动力学物理**（摩擦圆、载荷转移定量直觉、陀螺进动的通俗力学）——权威来源：物理教材、MSF/官方驾培进阶材料、车辆动力学入门。

### 御_二（对应：交通/驾驶模拟里的"读路"、行人-路口冲突；也可借拟真驾驶）
- **现稿**：FMCSA 大车盲区、内轮差量级、A柱 IIHS 实测、SMIDSY/LBFTS、四体系收敛、三身份德行、Dutch Reach、眼神+减速、一臂宽、斑马线、远光/喇叭。
- **本次加到**（"读者能看懂人为什么会漏掉人"）：
  - **内轮差 = off-tracking 几何**：车转弯时后轮循着比前轮更小的圈（转向几何/Ackermann 直觉），扫掠路径的宽度≈随轴距增长——把"1.5–2 米"从一个量级变成一个**能推的几何**。
  - **A柱遮挡 = 角遮挡**：一根柱子在某个方位角上张开的遮挡角，恰好能吞掉一整个行人；随车速/夹角变化。
  - **LBFTS 的知觉科学做深**：扫视采样（saccade）与注意瓶颈、非注意盲/变化盲、conspicuity＝自下而上的显著性 vs 自上而下的预期（"心智模型里没给摩托留位置"）——把"看了没看见"讲成一个**可理解的知觉机制**，而不是一句俗语。
  - 四体系收敛**升海拔**：讲清"四套互不相干的体系为何收敛到同一组动作"这件事本身就是最强论证。
- **补库**：`防御性驾驶与移动的德行.md`（52k）为主，需**补知觉科学**（LBFTS/inattentional blindness/conspicuity 的同行评审机制）与**转向几何/off-tracking**（交通工程/车辆几何入门）。

### 御_三（对应：铁路调度 / 空管 / 港口货运经营 / 城市交通经营 / 工厂产线经营）——**模拟游戏深度中心**
- **现稿**：包裹链路、最后一公里、四模式调度（BAP/NP难、闭塞/发车间隔、绿波相位差公式、民航分离标准/流量三层）、瓶颈/TOC/关键路径、海地机场排队论、流动>囤积、模拟游戏演习脚本。
- **本次加到**（"玩专业调度/经营模拟的读者能直接拿去用"）：
  - **排队论作统一数学**（本次最大的一块深度）：**Little 定律 L=λW**；**利用率–延迟的非线性**——当利用率 ρ 逼近 1，排队时间**爆炸式**上升（这就是为什么没有哪条线敢跑到 100% 满载、为什么"再挤一点点"会突然崩）；变动性（到达/服务的波动，Kingman 直觉）如何把延迟推高。这一套**同时统一了**港口泊位、铁路发车间隔、空管流量、匝道控制——**也正是御_四 人力流水线的同一条数学**（显式回扣深度脊）。
  - 每模式再下沉到"模拟游戏里认得出"的机制：铁路（闭塞分区↔发车间隔的构成拆解，回扣御_一 制动∝v²）、民航（分离最小值含尾流分类、流量管理三层、时刻协调谱）、港口（BAP/堆场/岸桥调度）、公路（信号配时三参数 + 绿波带宽/相位差、匝道控制的冲击波）。
  - **护栏不松**：始终写成"**你怎么组织这条流**"（能动），不写成"物流系统怎么运转"（旁观，会塌进"数"）；不越 `科普层级上限律`——讲机制，不写"如何把一群人拉成持续运行的调度组织"。
- **补库**：`运输调度基础.md`（48k）+ `应急后勤与资源流.md`（39k）为底，**必须新补排队论**（Little 定律、M/M/1 排队时延、利用率-延迟曲线、Kingman 公式的通俗直觉）——权威来源：运筹学/排队论教材、交通流理论；四模式的机制补深（铁路信号工程、ICAO/民航局分离与流量、集装箱码头运作、交通信号手册）。

### 御_四（对应：工厂产线/自动化/基地建造/殖民地经营里的"搬运-产线-吞吐"）——**实践高峰**

## 五、御_四 ＝ 实践高峰（专节）

御_四 要被当成**整部御的压轴与情感高点**来写，不是第四个并列章：

- **它是弱者可达性的顶点**：搬运/绳结/简单机械/人力流水线，全部零资源、零权威、未到驾龄也当下能做。前三章总有"知识+能练则练+其余当准备"的落差；到这里，落差归零——**讲的每一样，读者今天就能亲手做完**。这正好兑现全书"护住最弱读者"的伦理。
- **它是深度脊的合拢处**：御_一 教你一个人怎么动，御_三 教你怎么组织一条大流，御_四 让你**亲手把一条流攥在手里**——人链就是把"流"缩回你和身边十个人的尺度，用 Little 定律同一条数学（删空手往返＝提流率）。三 里那句"下一章你会亲手排一条人力流水线"，在这里被接住、被验证。
- **它是魂最实的落地**：移动用于守护，在这里不再是"让一脚、回个头"，而是"把一群人的力气临时编成一条线，把东西又快又稳送到位，然后散场"。**组织＝能力非归属**在这一章讲到最干净（临时编组、完成即散、不留名册不留队长）。
- **写法**：全御最厚、最操作、最有画面的一章；四段（一个人安全搬 → 用绳/棍/板/轮放大力气 → 排一条人力流水线）逐段往深、往手上落；收尾是全御的收尾，硬落、不鸡汤。

## 七、御专属硬约束（逐条核查）

- **律7（不动的硬线）**：全程民用。无任何军用载具/大战场/通用作战；坦克/军用无人机/军用空管/军事飞行管制**零出现**。即便某内容能用游戏复现，也不因此松动。（御_三 民航流量规定原文含"军事活动"一词——正文已剥离改写为"临时的空域调整"，重写沿用此处理。）
- **科普层级上限律**：讲机制、讲科普，不写"如何把一群人拉成持续运行的组织/调度队伍"。写到那种颗粒度即越界。
- **组织＝能力非归属**：人力流水线、物流协调都是**临时的、任务完成即散**；语气"因此你能做更多"，不是"所以你应该"；不要求读者加入/组建持续团队。
- **无敌人无英雄**：案例去主体化；不制造英雄。
- **减害立场（御_一 摩托）**：不鼓励无证/未达龄驾驶，但对"反正已经在骑"的读者不假装看不见；把安全驾驶讲到能建立习惯、护住这批最高危读者。**减害不是终点，合规才是终点**——到龄考证上牌、车不超标的话要说全。
- **弱者复核**：每条操作对"最弱体能+最少资源+最低社会权威+未到法定驾龄"复核。骑行/搬运/绳结/简单机械人人当下可练；摩托/汽车/物流按"知识+能练则练（含模拟器）+其余当准备"处理，不制造资源门槛（模拟器只说通用品类、给无设备读者留现实替代）。像射_一 那样：不给浅版，只在**特殊身体状态**处加修正（发育中身体配车/负重阈值）。
- **境外案例隔离**：中国本土案例只配本土合法解法；境外只取狭窄技术机制（调度参数、绳结力学、人体工学阈值、排队论、盲区几何），不作社会范本、不做跨国优劣排序。
- **用游戏的纪律（红线成文）**：只描述通用品类、**绝不点名**具体游戏/软件（背书+时效停服+分级+微交易风险）；模拟器只作"机制体验"不作"技能习得"；不荐 FPS；优先去主体化品类；每个"玩游戏当本周行动"须过真落地测试（玩它是否建立本章能力并迁移到现实）；屏幕动作不得挤掉更普惠的现实物理动作。
- **不生造/拼凑引用**：数字/人名/年份/机构名必先入库带来源；条件概率方向不得颠倒；不用学术腔掩盖空洞。已判否的数字（内轮差 70/90%、时速50一半致死、"2026后排新规"谣言等）**不得复活**。
- **否定式扫描**：定稿前把"不是/而不是/并非"尽量改肯定陈述。
